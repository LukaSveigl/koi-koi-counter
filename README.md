# Koi-Koi Counter

A mobile-first score counter and round tracker for the Japanese card game Koi-Koi (Hanafuda).

Built with Angular and Angular Material.

---

## Features

- 2-player Koi-Koi score tracking
- Configurable rulesets
- Multiple score presets
- Custom yaku values
- Oya selection
- Round-by-round score tracking
- Final game summary
- Mobile-first UI

---

## Tech Stack

- Angular
- Angular Signals
- Angular Material
- TypeScript
- SCSS

---

## Development

Install dependencies:

```bash
npm install
```

Run locally:

```bash
ng serve
```

Open:

```txt
http://localhost:4200
```

---

## Build

```bash
ng build
```

---

## Deployment (GitHub Pages)

Build for production:

```bash
ng build --base-href "/koi-koi-counter/"
```

The generated files are inside:

```txt
dist/
```

Deploy the contents to GitHub Pages.
